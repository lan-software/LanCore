# Documentation

LanCore documentation is organized using MIL-STD-498 (Military Standard for Software Development and Documentation) as the primary structure.

## MIL-STD-498 Documentation

The complete MIL-STD-498 document set is located in [mil-std-498/](mil-std-498/README.md). It contains 22 Data Item Descriptions covering:

- **Plans:** [SDP](mil-std-498/SDP.md), [SIP](mil-std-498/SIP.md), [STrP](mil-std-498/STrP.md)
- **Requirements:** [OCD](mil-std-498/OCD.md), [SSS](mil-std-498/SSS.md), [SRS](mil-std-498/SRS.md), [IRS](mil-std-498/IRS.md)
- **Design:** [SSDD](mil-std-498/SSDD.md), [SDD](mil-std-498/SDD.md), [DBDD](mil-std-498/DBDD.md), [IDD](mil-std-498/IDD.md)
- **Testing:** [STP](mil-std-498/STP.md), [STD](mil-std-498/STD.md), [STR](mil-std-498/STR.md)
- **Manuals:** [SUM](mil-std-498/SUM.md), [SIOM](mil-std-498/SIOM.md), [SCOM](mil-std-498/SCOM.md), [COM](mil-std-498/COM.md), [CPM](mil-std-498/CPM.md), [FSM](mil-std-498/FSM.md)
- **Product:** [SPS](mil-std-498/SPS.md), [SVD](mil-std-498/SVD.md)
