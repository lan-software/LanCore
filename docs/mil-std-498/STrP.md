## 1. Scope

### 1.1 Identification

This Software Transition Plan (STrP) describes the plan for transitioning the **LanCore** software to the support organization.

### 1.2 System Overview

LanCore is an open-source project. The "support organization" is the open-source community and individual self-hosting operators.

## 3. Transition Overview

### 3.1 Transition Model

As an open-source project, LanCore transitions from development to support through:

1. **Code publication** — Source code available on GitHub
2. **Documentation** — MIL-STD-498 documentation set
3. **Docker images** — Pre-built container images on GHCR
4. **Community support** — GitHub Issues and Discussions

### 3.2 Support Responsibilities

| Responsibility | Owner |
|---------------|-------|
| Bug fixes and security patches | Core maintainers |
| Feature development | Community contributors |
| Deployment and operations | Self-hosting operators |
| User support | Community forums |
